Definitions

Power traits that represent magic in the Tower Builders Setting have some common features. The following terms are helpful to create a common language among the players.

Result
The intended purpose of any use of magic. After casting, the system result of the effect of a successful spell such as an asset or complication.
Magical Power
A Power or Ability that follows the rules of magic for the setting.
Magical Power Set
A Power Set that collects a set of magical powers and abilities that represent a use of the same chracter feature, training, or magical item.
Improvisational Power Set
A Material Components Power Set or Taxonomy Power set. Improvisational Power Sets provide flexible use of magic and may have the modify power to modify magical abilities.
Taxonomy
A description of the elements of the world and how they interact both naturally and unnaturally. Alternate terms include Language, Pantheon, Alphabet, and Ontology.
Taxonomies Set
A power set that collects all of the powers and specialties that represent a character’s taxonomy powers. Will usually have a Limit that causes Shut Down when the caster cannot cast.
Material Components Set
A power set that collects all of the powers and specialties that represent a character’s material components powers. Will usually have a Limit that describes practicalities of working with large amounts of materials.
Taxonomy Power
A power that represents the ability to create magical results or act on targets that fit the taxonomy’s worldview according to the [The Laws of Magic].
Reference
n. A feature of the world an Improvisational Power can influence or create. A reference is not necessarily fundamental to the taxonomy (as an element is), or available as an element trait. For example: the Taxonomy Forest Life can create results on snails because they are a part of the forests it describes even though snails are not an Element of Forest Life. v. The act of using magic on a reference.
Element
1. A specialty of a taxonomy power that is included in dice pools when referencing that element of a taxonomy. 2. A major component of a Taxonomy that defines and creates the taxonomy’s worldview. Many alternate names exist including: Glyph, Deity, Symbol, Rune, Word.
Target
The subject of a spell. This can be another spell. See [Targeting].
Casting
The specific activity(s) a character takes to create a magical result.
Casting Implement
A wand, staff, amulet, or other physical object that assists in casting. Often a signature asset.
Magic Dice
The dice in a pool provided by magical abilities, assets, distinctions, taxonomies, or elements.
Casting Dice
Any dice in a pool that represent the mundane activities a character is undertaking to create a magical result. Usually dice from the prime sets attributes and training.
Targeting Method
The method a caster is using to deliver magic to a target. Usually one of Touch, Bolt, Focus, or Affinity. See [Targeting].
